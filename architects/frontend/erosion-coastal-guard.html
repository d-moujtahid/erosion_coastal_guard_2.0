Contexte & Méthodologie

Le présent rapport documente le recul du trait de côte sur la façade littorale d'Agadir et Taghazout entre 2016 et 2026. Les données ont été collectées par relevés GPS terrain à fréquence semestrielle, combinés à une analyse photogrammétrique aérienne annuelle. La distance de recul entre deux points homologues est calculée via la formule de Haversine, garantissant une précision métrique sur les courtes distances côtières. Le système de monitoring s'appuie sur une base de données MySQL 8.0, une API REST Node.js/Express et un dashboard interactif React/Leaflet.

Résultats Principaux
Recommandations
  • 🚨 Priorité 1 : Déploiement immédiat d'enrochements et brise-lames sur le segment Taghazout Nord (recul de 47.3 m en 10 ans). Intervention d'urgence recommandée avant l'hiver 2026.
  • 🏗️ Priorité 2 : Rechargement sédimentaire artificiel sur la plage d'Agadir Centre (Zone 3) pour compenser le déficit de 38.1 m observé depuis 2016.
  • 📡 Priorité 3 : Installation de 4 capteurs GPS permanents sur les zones critiques pour passage d'une surveillance semestrielle à un monitoring temps réel continu.
  • 📊 Priorité 4 : Intégration de modèles de prédiction par régression linéaire pour anticiper le trait de côte à l'horizon 2030, sur la base des tendances 2016–2026.
  • 🤝 Priorité 5 : Partage des données brutes avec le Ministère de l'Équipement et l'Agence du Bassin Hydraulique du Souss-Massa pour coordination régionale.
